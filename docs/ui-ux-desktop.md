# UI/UX — Desktop (Tauri)

## Общая концепция
- Стиль: тёмный, минималистичный, вдохновлён Telegram Desktop + современные элементы 2025–2026
- Основная тема: очень тёмно-серый (#0F1117) + сине-голубой акцент (#5E9FFF)
- Опционально: неоновый режим (#00D4FF + glow-эффекты)
- Split-view по умолчанию (левый список чатов, правый — текущий чат)
- Адаптивность: при ширине окна < 900px — мобильный стиль (один экран)

## Цветовая палитра (основная)

- Фон основной:           #0F1117
- Фон карточек/сообщений: #16181F
- Акцент:                 #5E9FFF
- Неон (опционально):     #00D4FF
- Текст основной:         #E8EEF7
- Текст вторичный:        #A0AEC0
- Свои сообщения:         #2B5278
- Чужие сообщения:        #1E293B
- Онлайн/успех:           #34C759
- Ошибка/оффлайн:         #FF3B30

## Структура экрана

### 1. Главное окно (Main Window)

- **Левая панель** (ширина ~320–380 px, фиксированная)
  - Верх: аватар текущего пользователя + ник + статус (онлайн/оффлайн)
  - Поиск по чатам/контактам (с иконкой лупы)
  - Список чатов/групп (вертикальный скролл)
    - Аватар (круглый, 48×48 px)
    - Название + последнее сообщение (2 строки)
    - Время / непрочитанные счётчики (зелёный кружок)
  - Нижняя часть: кнопка «+» (новый чат / группа)

- **Правая часть** (основной контент)
  - Если чат не выбран → приветственный экран / пустой чат
  - Если чат выбран → заголовок чата + список сообщений + поле ввода

### 2. Экран чата

- **Заголовок чата**
  - Аватар + имя + статус (онлайн / last seen / typing…)
  - Справа: ⋮ меню (инфо, поиск по чату, очистить историю, выйти из группы и т.д.)

- **Область сообщений**
  - Пузырьки с закруглёнными углами (16–20 px)
  - Свои — справа, тёмно-синий (#2B5278)
  - Чужие — слева, тёмно-серый (#1E293B)
  - Время — маленькое, серое, снизу справа от пузырька
  - Статусы: ✓ (отправлено), ✓✓ (доставлено), ✓✓✓ (прочитано) — зелёные
  - При скролле к непрочитанным — лёгкая синяя полоска слева

- **Нижняя панель ввода**
  - Поле ввода (растягивается по высоте до 5 строк)
  - Слева: микрофон (голосовое), скрепка (файл/фото/документ)
  - Справа: эмодзи-пикер
  - При фокусе → кнопка отправки становится #5E9FFF и увеличивается

### 3. Экран добавления контакта / группы

- Модальное окно или боковая панель справа
- Варианты:
  - «Сканировать QR» (камера или загрузка изображения)
  - «Ввести ключ / ссылку» (большое текстовое поле)
  - После ввода ключа → появляется поле «Введите 12-значный код»
    - Автоподсветка при вводе (зелёный — верно, красный — ошибка)
    - Таймер «Код истекает через 15:00»

### 4. Настройки (Settings)

- Отдельное окно или боковая панель
- Вкладки:
  - Аккаунт (публичный ключ, QR для добавления)
  - Внешний вид
    - Тема: Тёмная (по умолчанию)
    - Цвет акцента (палитра или HEX)
    - Неоновый режим (переключатель)
    - Кастомный фон чата (градиент / паттерн / отключено)
  - Приватность (таймер самоуничтожения, panic wipe)
  - Хранилище (очистка кэша, лимит mailbox)

### 5. Дополнительные элементы

- Typing индикатор: «Имя печатает…» + три пульсирующие точки (в неоне — cyan)
- Неоновый режим: glow на активных кнопках, индикаторах, новых сообщениях
- Анимации: плавное появление сообщений снизу, лёгкое масштабирование кнопки отправки
- Клавиатурные сокращения: Ctrl+K — поиск, Esc — закрыть модалку, Ctrl+N — новый чат
