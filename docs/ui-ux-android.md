# UI/UX — Android (Jetpack Compose)

## Общая концепция
- Стиль: максимально близок к Telegram Android
- Тёмная тема по умолчанию
- Material You элементы + кастомные цвета
- Поддержка жестов, swipe, long press
- Учёт энергосбережения и маленьких экранов

## Цветовая палитра
(та же, что на Desktop)

- Фон: #0F1117
- Surface: #16181F
- Primary: #5E9FFF
- Neon (опционально): #00D4FF
- On-surface: #E8EEF7
- On-surface-variant: #A0AEC0

## Структура экранов

### 1. Главный экран (Chats List)

- TopAppBar:
  - Заголовок: NeoChat
  - Справа: поиск (лупа), меню (⋮)

- FloatingActionButton (+): новый чат / группа
- Список чатов (LazyColumn):
  - Карточка чата:
    - Аватар слева (круглый)
    - Название + последнее сообщение
    - Время / непрочитанные (зелёный бейдж)
  - Swipe actions:
    - Свайп вправо → архивировать
    - Свайп влево → удалить / закрепить

### 2. Экран чата (Chat Screen)

- TopAppBar:
  - Назад ←
  - Аватар + имя + статус (онлайн / typing)
  - Справа: ⋮ (инфо, медиа, поиск по чату)

- Сообщения (LazyColumn, reverseLayout = true):
  - Пузырьки с закруглением 16–20 dp
  - Свои — справа, #2B5278
  - Чужие — слева, #1E293B
  - Время и статусы доставки — маленькие, снизу
  - При долгом нажатии → контекстное меню (ответить, копировать, удалить…)

- Bottom bar ввода:
  - TextField (растягивается по высоте)
  - Иконки слева: микрофон, скрепка
  - Справа: эмодзи → отправка
  - При фокусе → кнопка отправки становится Primary-цветом

### 3. Добавление контакта / группы

- BottomSheet или отдельный экран
- Варианты:
  - Сканировать QR (камера)
  - Ввести публичный ключ / ссылку
  - После ввода → поле «12-значный код»
    - Автоматическая проверка после каждого символа
    - Клавиатура только буквы + цифры
    - Зелёная/красная подсветка

### 4. Настройки (Settings Screen)

- Navigation Drawer или BottomNavigation + отдельный экран
- Секции:
  - Аккаунт (мой QR, публичный ключ)
  - Внешний вид
    - Неоновый режим
    - Выбор акцентного цвета
    - Тема (только тёмная пока)
  - Приватность
  - Хранилище и данные
  - О приложении

### 5. Особенности мобильной версии

- Bottom sheets для большинства модальных действий
- Pull-to-refresh для синхронизации mailbox при наличии интернета
- Индикатор подключения к пирам (маленькая иконка в TopAppBar)
- Foreground-уведомление при активном Bluetooth-relay
- Биометрия / PIN для открытия приложения (опционально)
- Адаптивные размеры: большие кнопки, читаемый текст
